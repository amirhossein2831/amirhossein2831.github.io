Authenticator

Notification Sender
The Go-based Notification Microservice project is designed to efficiently handle email and SMS notifications, providing a multi-driver approach with support for several SMS providers. This project is a microservice that focusing on scalability and reliability.
  • Multi-Driver Notification System: The service supports sending emails via SMTP and SMS notifications through multiple providers, including SMS-ir, Kavenegar, and Twilio, using a builder pattern for flexibility.
  • SMS Queuing with RabbitMQ: RabbitMQ is used to queue SMS messages to process them concurrently for fast and reliable delivery.
  • Scalable Architecture: The project handles high volumes of notifications using RabbitMQ and multiple workers for fast processing. Dockerized for easy deployment, it ensures seamless scalability and integration across environments.
